Hired Work

Tunche

2018-01-08
2021-08-19

This was my last project before leaving Leap Game Studio to come to the UK to study at Newcastle University.

It is 2D side-scrolling brawler with roguelike elements. The target systems where the PS4 and PC, we had already approached Sony and got some feedback from them, so we were excited about keeping the good work on this one.

This project had 2 programmers nearly from the start, I helped on most of the battle system, the level editor tool, the enemies wave editor, enemies AI, animation events, day/night cycle shaders, local multiplayer support and the dialogue system.

This was a project I was really sad of leaving because we got positive feedback from different sources, not that the game was perfect or amazing, but that it had potential, and that drove us further every day.

The game is still in development and in constant improvement.

Technology Used

  • Texture Packer
